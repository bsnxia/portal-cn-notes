# portal-cn-notes

本仓库用于归档与发布多个独立 HTML 页面。这些页面以笔记、信息整理或小型工具页面为主，内容涵盖技术参考、学习记录与实用功能。

仓库不针对任何特定域名或具体网站，仅作为静态页面的集合与发布点。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库导航页（可选）
├── pages/             # 主要页面存放目录
│   ├── example-1.html
│   └── example-2.html
└── assets/            # 页面共用资源（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，每个页面功能独立，互不依赖。
- `assets/`：存放多个页面可能共用的静态资源，便于统一维护与更新。

## 页面归档说明

每个 HTML 页面均为自包含的独立文档，可直接在浏览器中打开使用。页面之间没有强依赖关系，可以单独引用或部署。

归档内容主要来自：
- 个人学习笔记整理
- 日常实用工具页面
- 技术参考与速查表
- 其他零散但可独立展示的信息

部分页面可能包含外部资源引用（如 CDN 库），但会尽量保持轻量与离线可用性。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 新增页面时，建议在 `pages/` 下创建新文件，并更新 `index.html`（如有）中的导航链接。
- 欢迎通过 Issues 提出改进建议或报告页面问题，但请勿提交与仓库主题无关的内容。
- 仓库不保证所有页面长期可用或内容持续更新，部分页面可能因信息过时而移除。

## 许可

本仓库内容仅供个人学习与参考使用。如需引用或转载，请注明出处。
